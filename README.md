# 🕹️ Random Seats: Classroom Arcade
ระบบจัดโต๊ะเรียน / สุ่มที่นั่งนักเรียน สำหรับครู

## 🎯 ฟีเจอร์หลัก
- จัดผังห้องเรียนเองได้ (กระดาน, โต๊ะครู, โต๊ะนักเรียน 1–8 คน)
- เพิ่มรายชื่อนักเรียน 20–50 คน
- กด SPIN สุ่มที่นั่งได้ทันที
- จัดที่นั่งเองได้โดยกดที่นั่งที่ต้องการ
- ล็อกที่นั่งที่ไม่ต้องการสุ่ม
- พิมพ์ผังที่นั่งออกกระดาษได้
- บันทึกข้อมูลอัตโนมัติในเบราว์เซอร์ รองรับหลายห้องเรียน

## 🚀 วิธีใช้งาน
เปิดไฟล์ `classroom-seating-arcade.html` ในเบราว์เซอร์ได้เลย  
ไม่ต้องติดตั้งโปรแกรมเพิ่มเติม
